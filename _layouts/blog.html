About {{ site.title }}

RL4RL is a project designed to encourage the use of Reinforcement Learning for Real Life problems. Specifically for data in which decisions are made in sequences that lead towards a long term outcome. This website showcases some applications from a range of domains to help demonstrate how Reinforcement Learning can be applied in this way

We hope that by making the method more accessible, more and more uses will come from this work. This is achieved by:

  1. Defining the process required to model the environment as a Markov Decision Process (MDP).
  2. Introducing an R/Python package that can be applied on the MDP for training an agent.
  3. Writing clear educational examples which are added to the documentation to demonstrate the possible use cases for applying Reinforcement Learning to real-life tasks.
  4. Encouraging a community to showcase their work and novel applications, further increasing the number of use cases.

The package and documentation is made publicly available and free to use but request that you cite our accompanying paper ..... REF.....

This project was founded by Philip Osborne, a PhD student of the University of Manchester's AI Systems Lab.
