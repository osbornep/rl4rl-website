About {{ site.title }}

RL4RL is a project designed to encourage the use of Reinforcement Learning for Real Life problems. Specifically for tasks where decisions are made in sequences that lead towards a long term outcome. This website showcases some applications from a range of domains to help demonstrate how Reinforcement Learning can be applied in this way

We hope that by making the method more accessible, more and more uses will come from this work. To achieve this, we created R and Python packages that can be imported to make applying Reinforcment Learning even easier. Furthermore, the demonstrations made available here show exactly what is required to model the data correctly before inputting into the functions.
